# Recruiter outreach draft

Create concise outreach from `MASTER_CV`, `JOB_DESCRIPTION`, `JOB_METADATA`, `CONTACT_CONTEXT`. Candidate claims: master CV only. Contacts must be verified public entries with source URLs.

## Channels

- Verified professional email → email.
- Verified LinkedIn profile → LinkedIn message.
- Both → both.
- Neither → both with `[Recruiter Name]`, `[Email]`, `[LinkedIn URL]`.
- One only → omit unavailable channel; no placeholder for it.

Never infer email/profile patterns, use personal contacts, or claim prior relationship. Draft only; never send or connect.

Write like a person asking one specific professional question: direct opening, concrete proof point, one CTA. Remove generic praise, sales language, filler, forced trios, chatbot commentary, and repeated sentence shapes. Do not add a separate humanization pass.

Use exact vacancy title and one strongest supported proof point, preferably metric/technical outcome. Connect it to one role requirement. End with one low-friction request. Use verified name naturally. No superficial personalisation, generic praise, unsupported company facts, attachments, multiple achievements, or multiple asks.

Return outreach Markdown only; no analysis, score, alternatives, or code fence.
